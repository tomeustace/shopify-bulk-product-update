# hhpup

## Description

windows task calls app on a daily basis.  The task opens browser and navigates to merge url to trigger update.
log file is populated with what changed.  log file is in json format so can be viewed in app.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
